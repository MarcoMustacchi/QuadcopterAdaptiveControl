<h1 align="center">Computer Vision - Laboratory Activities <br> UniPd</h1>

## Build
Clone this repository
```
git clone https://github.com/MarcoMustacchi/ComputerVision
cd ComputerVision
cd Lab1
```

Execute the following commmands
```
mkdir build
cd build
cmake ..
make
```

## Run
```
./Lab1
```
